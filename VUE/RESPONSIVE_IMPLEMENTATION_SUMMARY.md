- 响应式间距：`.gap-responsive`, `.p-responsive-lg`, `.p-responsive-md`, `.p-responsive-sm`, `.px-responsive`

## 响应式设计最佳实践

### 1. 移动优先方法
- 从最小屏幕开始设计
- 逐步增加功能和复杂性
- 使用 `min-width` 媒体查询

### 2. 灵活的布局
- 使用 Flexbox 和 Grid
- 避免固定宽度
- 使用相对单位（rem、em）

### 3. 触摸友好的界面
- 最小触摸目标大小：44x44px
- 充足的间距
- 清晰的交互反馈

### 4. 图片优化
- 使用 `max-width: 100%`
- 使用 `object-fit` 处理图片
- 响应式图片尺寸

### 5. 字体大小
- 最小字体大小：12px
- 使用相对单位
- 确保可读性
