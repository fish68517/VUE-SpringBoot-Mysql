| 数据类型       | 来源渠道                   | 说明                    |
| ---------- | ---------------------- | --------------------- |
| **课程资料数据** | 学校课程教学大纲、教材内容摘要、学生上传资料 | 属于校内真实数据，可通过Excel导入   |
| **学习行为数据** | 学生答题记录、浏览历史、学习时长       | 系统运行过程中自动产生           |
| **视频资源数据** | Bilibili/慕课网/公开课程（教育类视频）    | 仅抓取标题、简介与播放量等公开字段（合规的） |
| **题库数据**   | 教师提供的历年试题或学校练习题        | 经过脱敏处理后导入系统   可通过Excel导入        |
