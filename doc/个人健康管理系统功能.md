- # 技术方案

1: springboot 3.X + Vue + Mysql

2:  springboot 不需要复杂的安全认证，不需要安全相关的框架，密码不需要加密，

支持跨域访问，所以需要保证所有的接口都能访问，也不需要JWT token等认证 。

3： VUE ui框架设计： 用户端登录后，采用顶部导航菜单栏+对应内容的结构

医生和管理登录后，采用 侧边导航菜单栏+对应内容结构

注册和登录页面不需要 菜单栏，且根路径直接定向到登录页面。

4： 关于注释和调试日志以及web网页字符，全部采用简体中文显示
