首页

# 管理员端

先完成管理员端功能，技术如下

* 后端不要使用 JWT、Spring Security、Sa-Token、RBAC 等权限框架。数据库mysql
* 登录密码不加密，直接明文校验。
* 图片上传后直接保存到 Spring Boot 项目根目录 image 文件夹下。
* 数据库中只保存图片文件名或相对路径，不保存图片二进制。
* 需要配置静态资源映射，使 /image/** 可以直接访问图片。
* 需要配置全局 CORS，允许所有前端来源访问所有接口和图片资源。
* 前端使用 Vue 3 + Vue Router + Pinia + Axios + Element Plus。
* 管理员采用左侧边栏 + 内容区布局。
* 登录后跳转到不同布局页面。
* 页面风格简洁，适合本科毕业设计展示。
* 代码结构保持清晰，按 controller、service、mapper、entity、dto、vo、config 分层。
* 优先保证系统可运行、页面可访问、接口可联调，不要过度设计。
* 页面所有的文本采用：中文简体，代码注释和调试日志也采用 中文简体
