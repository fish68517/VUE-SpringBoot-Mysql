# 基于 Spring Boot + Vue 的沈阳音乐节管理系统设计与实现

## 1. 系统角色
- 普通用户（游客 / 注册用户）
- 管理员（后台管理）

## 2. 核心功能模块

### （一）用户与权限管理模块
- 用户注册 / 登录（手机号 + 验证码登录 或 账号密码登录）
- 实名认证（购票必填身份证号，用于防黄牛、一人一票）
- 个人信息管理：头像、昵称、手机号、邮箱、收货地址
- 登录日志记录（可选）
- 个人中心：我的订单、我的电子票、我的收藏、我的积分

### （二）音乐节资讯展示模块（突出沈阳地域特色）
- 音乐节主页：轮播大图、倒计时、沈阳城市宣传片、音乐节主题介绍
- 演出日程（多舞台时间轴）
- 演出艺人列表（可点击查看艺人详情 + 沈阳本土乐队重点标注）
- 沈阳天气实时查询接口（对接中国天气网或墨迹天气 API，显示音乐节期间逐日天气 + 穿衣建议）
- 交通指南、停车攻略、沈阳地铁/公交推荐
- 入场须知、防疫要求、安检须知

### （三）在线购票模块（无座号 → 分区限流 + 实名制）

**核心调整（按增补要求）：**
- 取消座位选座，改为“分区购票”（看台A区、B区、站区、前排VIP区等）
- 每个分区设置最大容纳人数（限额，例如 A 区最多 3000 人）
- 购票必须填写身份证号，同一身份证号只能购买 1 张票（防黄牛）
- 支持一人帮多人购票（最多 4 人），但每人都要填写身份证号
- 无退票功能（售出概不退换，仅支持改期如遇疫情）

**购票流程：**
1. 选择场次  
2. 选择分区  
3. 填写购票人实名信息  
4. 下单  
5. 模拟支付（跳转支付成功页）  
6. 生成电子票（带二维码 + 身份证后四位）

**我的电子票：** 可查看、下载 PDF（使用 pdf-make 或 html2canvas 生成）

### （四）音乐节周边商品售卖模块
- 商品分类（T恤、卫衣、帽子、徽章、应援棒、光盘等）
- 商品上下架、库存管理
- 商品详情多图、规格（颜色、尺码）、价格
- 加入购物车 → 结算 → 填写收货地址 → 模拟支付
- 订单管理（物流状态模拟：已支付 → 已发货 → 已完成）
- 无退单功能（售出概不退换）

### （五）内容发布与用户互动模块
**管理员发布：**
- 新闻公告
- 音乐节攻略
- 交通住宿推荐
- 沈阳美食推荐

**用户互动：**
- 资讯点赞、收藏、评论
- 音乐节整体收藏功能（“加入我的关注”）
- 评论区楼中楼回复（可选）

### （六）盛京文化沉浸式互动模块
**功能名称：** 沈阳音乐节“打卡盛京”积分任务

**技术亮点：**
- 前端：Vue3（如“沈阳本土乐队舞台”“盛京夜市美食摊”“中街步行街快闪点”等）
- 用户在指定位置打开网页 → 上传合影照片 → 后台审核通过 → 获得积分

**积分规则引擎（Spring Boot 实现）：**
- 与沈阳本土乐队海报合影 +30 分
- 穿满族服饰打卡 +50 分
- …（可自行扩展）

**积分商城（可选后续扩展）：** 可用积分兑换周边小礼品

> **答辩亮点：** 真正做到“沈阳音乐节”而不是“通用音乐节”，地域文化沉浸式体验，几乎无雷同项目！

### （七）后台管理系统（Admin）
**仪表盘（ECharts）**
- 日订单量趋势（票务 + 商品）
- 用户增长曲线
- 分区售票热力图（哪个区卖得最快）
- 商品销量排行 TOP10
- 打卡任务完成统计

**功能模块：**
- 用户管理（封禁、查看实名信息）
- 票务管理（分区库存设置、价格设置）
- 商品管理（上下架、库存）
- 订单管理（票务订单 + 商品订单 + 导出 Excel）
- 积分任务管理
  - 新增/编辑/上下架任务
  - 设置任务分值、有效期、打卡位置（经纬度范围）
  - 批量审核用户上传的打卡照片（通过/拒绝 + 原因）
- 积分商城管理
  - 商品上下架、库存设置、积分价格
  - 兑换订单管理（确认发货、填写物流单号、导出 Excel）
- 积分流水查询（支持按用户、时间、任务类型筛选）
- 用户积分管理
  - 手动加减积分（用于活动补偿）
  - 封禁用户积分功能（恶意刷分）

### （八）积分商城
**位置：** 个人中心 → 我的积分 → 积分商城入口

**功能清单：**
- 商品类型（全部为沈阳音乐节限定，突出稀缺性）

**实体周边示例：**
- 沈阳本土乐队签名海报（限量 100 张） → 800 分
- 满族旗头造型应援棒（荧光） → 520 分
- “盛京 4643”城市徽章套装（含沈阳方言元素） → 360 分
- 音乐节限定 T 恤（不同乐队款） → 600~900 分
- 实体门票升级券（站区 → VIP 区） → 1500 分（最重磅）

**虚拟权益示例：**
- 艺人见面会抽奖机会 ×1 → 300 分
- 前排 VIP 早鸟通道资格 → 1200 分
- 官方摄影师跟拍 15 分钟（现场兑换） → 2000 分

**兑换流程：**
用户选择商品 → 检查积分是否足够 → 填写收货地址/选择场次 → 提交兑换申请 → 管理员后台审核 → 积分扣除 → 发放实体券/物流发货

**其他功能：**
- 库存实时显示 + 倒计时（如：“签名海报 剩余 3/100”）
- 兑换记录：可查看历史兑换订单、物流状态

## 4. 技术栈

### 后端
- Spring Boot 3.x + Spring Web MVC + Spring Data JPA
- MySQL 8 + Lombok + Maven
-不允许使用 Springboot 内置的安全组件 

### 前端
- Vue3（Composition API + script setup）
- Pinia（用户信息、购物车、打卡状态）
- Vue Router
- Element Plus + Arco Design（部分组件）
- Vite 5
- Axios + 拦截器
- ECharts 5
- html2pdf.js（电子票导出）
- 天气 API（中国天气网或心知天气）
