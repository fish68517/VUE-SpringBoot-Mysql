# 响应式设计测试文档

## 测试目标

验证广州旅游管理系统在不同屏幕尺寸和设备上的响应式表现，确保用户体验一致。

## 测试环境

### 浏览器

- Firefox 最新版本
- Safari 最新版本
- Edge 最新版本

### 测试工具
- Chrome DevTools 响应式设计模式
- Firefox 响应式设计模式
- 实际设备测试

## 附录

### 如何使用 Chrome DevTools 进行响应式测试

1. 打开 Chrome 浏览器
2. 按 F12 打开开发者工具
3. 按 Ctrl+Shift+M (Windows) 或 Cmd+Shift+M (Mac) 打开响应式设计模式
4. 在顶部选择设备或自定义分辨率
5. 测试页面在不同分辨率下的表现

### 常用测试分辨率

```
超大屏: 1920x1080, 1366x768, 1280x720
大屏: 1024x768, 1024x600, 900x600
中屏: 768x1024, 640x960, 600x800
小屏: 480x800, 414x896, 390x844, 375x667
超小屏: 360x640, 320x568
```
