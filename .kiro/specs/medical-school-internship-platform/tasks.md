# 医校联动实习平台 - 实现计划

## 实现任务清单

- [x] 1. 项目初始化与基础框架搭建




  - 创建 SpringBoot 项目结构，配置 pom.xml 依赖（Spring Data JPA, MySQL Driver, Lombok, Validation）
  - 配置 application.yml 数据库连接和 JPA 属性
  - 创建统一的 API 响应格式和异常处理类
  - _需求: 1.1, 1.6_

- [x] 2. 数据库设计与实体类创建





  - 创建 User、Organization、Post、Application、Internship、WeeklyReport、Evaluation、Notification 实体类
  - 配置实体类的 JPA 注解和关系映射
  - 创建所有实体对应的 Repository 接口
  - _需求: 1.1, 2.1, 3.1, 4.1, 5.1, 6.1_

- [x] 3. 用户认证与组织隔离功能






























  - 实现用户注册接口（POST /api/users/register），支持选择身份类型和组织
  - 实现用户登录接口（POST /api/users/login），返回用户信息和组织信息
  - 实现获取个人信息接口（GET /api/users/profile）
  - 实现更新个人信息接口（PUT /api/users/profile）
  - 创建会话拦截器，验证每个请求的用户身份和组织隔离
  - _需求: 1.1, 1.2, 1.3, 1.4, 1.5, 1.6_

- [x] 4. 组织管理功能（系统管理员）





  - 实现获取所有组织接口（GET /api/organizations）
  - 实现创建组织接口（POST /api/organizations）
  - 实现更新组织信息接口（PUT /api/organizations/{id}）
  - 实现获取组织内用户接口（GET /api/organizations/{id}/users）
  - _需求: 1.1_

- [x] 5. 岗位管理功能





  - 实现获取岗位列表接口（GET /api/posts），根据用户身份进行数据隔离
  - 实现创建岗位接口（POST /api/posts），医院管理员创建，设置面向学校范围
  - 实现更新岗位接口（PUT /api/posts/{id}），医院管理员编辑
  - 实现上下架岗位接口（PUT /api/posts/{id}/status），医院管理员操作
  - 实现获取岗位详情接口（GET /api/posts/{id}）
  - 实现岗位名额检查逻辑，名额已满时禁止申请
  - _需求: 2.1, 2.2, 2.3, 2.4, 2.5_

- [x] 6. 岗位申请与审批功能





  - 实现提交申请接口（POST /api/applications），学生提交，记录申请理由
  - 实现获取申请列表接口（GET /api/applications），根据用户身份返回相应数据
  - 实现获取申请详情接口（GET /api/applications/{id}）
  - 实现学校审批接口（PUT /api/applications/{id}/school-review），学校管理员审批
  - 实现医院审批接口（PUT /api/applications/{id}/hospital-review），医院管理员审批
  - 实现审批通过时自动生成实习记录的逻辑
  - _需求: 3.1, 3.2, 3.3, 3.4, 3.5_

- [x] 7. 实习过程管理功能





  - 实现获取实习记录列表接口（GET /api/internships）
  - 实现获取实习记录详情接口（GET /api/internships/{id}）
  - 实现提交周记接口（POST /api/internships/{id}/weekly-reports），学生提交
  - 实现获取周记列表接口（GET /api/internships/{id}/weekly-reports）
  - 实现批阅周记接口（PUT /api/internships/{id}/weekly-reports/{reportId}/review），老师批阅、评分、打回
  - 实现提交评价接口（POST /api/internships/{id}/evaluations），支持老师评学生和学生评老师
  - 实现获取评价列表接口（GET /api/internships/{id}/evaluations）
  - _需求: 4.1, 4.2, 4.3, 4.4, 4.5_

- [x] 8. 数据统计功能





  - 实现学校数据统计接口（GET /api/statistics/school），返回申请人数、通过人数、实习中人数、完成率等
  - 实现医院数据统计接口（GET /api/statistics/hospital），返回岗位发布数、投递数、录用数等
  - 实现统计数据的实时更新逻辑
  - _需求: 5.1, 5.2, 5.3, 5.4_

- [x] 9. 通知功能





  - 实现获取通知列表接口（GET /api/notifications）
  - 实现标记通知为已读接口（PUT /api/notifications/{id}/read）
  - 实现通知生成逻辑（申请审批结果、周记退回、任务截止提醒）
  - _需求: 6.3, 6.4, 6.5_

  - 创建全局状态管理（Pinia 或 Vuex），存储用户信息和组织信息
